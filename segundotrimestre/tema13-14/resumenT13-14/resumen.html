Tema 13-14: Transformaciones XSL XSLT

¿Qué es XSLT?

XSLT (eXtensible Stylesheet Language Transformations) es un lenguaje basado en XML que permite transformar documentos XML en otros formatos como HTML o XML.

Características de XSLT

  • Lenguaje declarativo basado en reglas.
  • Transforma documentos XML en otros formatos.
  • Permite generar HTML dinámico.
  • Utiliza XPath para seleccionar y manipular datos.

Ejemplo de Transformación con XSLT

        <xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform" version="1.0">
         <xsl:template match="/">
         <html>
            <body>
                <h1><xsl:value-of select="/concesionario/coche/marca"/></h1>
            </body>
         </html>
            </xsl:template>
            </xsl:stylesheet>

Herramientas para XSLT

  • Oxygen XML Editor: Editor XML con soporte para XSLT.
  • Altova XMLSpy: Herramienta avanzada para manipulación XML.
  • Saxon: Procesador XSLT de código abierto.
  • Xalan: Implementación de XSLT en Apache.
