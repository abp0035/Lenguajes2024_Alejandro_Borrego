Tema 11:Validación XML Schema

¿Qué es XML Schema?

XML Schema (XSD) es un lenguaje basado en XML que permite definir la estructura de un documento XML y validar su contenido.

Propósito de XSD

  • Define los elementos y atributos permitidos en un XML.
  • Especifica el número y orden de los elementos secundarios.
  • Determina los tipos de datos que pueden usarse.
  • Permite establecer valores predeterminados y restricciones.

Ejemplo de Estructura XSD

<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema">
    <xs:element name="persona">
        <xs:complexType>
            <xs:sequence>
                <xs:element name="nombre" type="xs:string" />
                <xs:element name="edad" type="xs:integer" />
            </xs:sequence>
        </xs:complexType>
    </xs:element>
</xs:schema>

Asociación de un XML con XSD

<persona xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xsi:noNamespaceSchemaLocation="persona.xsd">
    <nombre>Juan</nombre>
    <edad>30</edad>
</persona>
